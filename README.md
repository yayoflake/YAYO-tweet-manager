# YAYO tweet manager

트위터에서 받은 내 백업 데이터를 보기 편하게 탐색하고, 티스토리 블로그 등에 옮기기 위한 개인 도구입니다.
기본적으로는 티스토리 업로드용으로 만든 앱입니다만 md, txt, pdf 등의 여러가지 포맷으로 내보내는 기능도 붙였으니 필요하면 사용해 주세요.

실행할 때 Node.js가 필요합니다. 아직 설치하지 않았다면 **[nodejs.org](https://nodejs.org)** 에서 설치해 주세요.


## 실행하는 방법

### 1. 내 트위터 아카이브 백업 받기

1. 트위터 → 설정 및 개인정보 → 계정 → 내 데이터 기록 파일 다운로드
2. 메일로 날아온 zip 파일 받기
3. 압축 풀기

### 2. 폴더 배치

트위터 아카이브 데이터 폴더 안에 YAYO tweet manager 폴더를 넣으세요. 

```
📁 트위터 아카이브 폴더
├── 📁 assets              
├── 📁 data    
├── 📄 Your archive.html 
└── 📁 YAYO tweet manager  ← 이 프로그램
```

### 3. 실행

YAYO tweet manager 안의 **`start.bat`** 파일을 더블클릭하면 브라우저가 자동으로 열립니다.


## 사용하는 방법

### 트윗 찾기

왼쪽 필터 패널에서 원하는 조건으로 트윗을 걸러냅니다.

- **키워드** — 본문에서 단어 검색 (`-단어` 형식으로 제외 가능)
- **날짜** — 기간을 지정하거나 특정 날짜만 보기
- **리트윗 / 마음에 들어요 수** — 일정 수 이상 반응이 있는 트윗만 보기
- **미디어** — 사진/동영상이 있는 트윗만 따로 보기
- **태그** — 일반 트윗, 인용 트윗, 익명 질문 등 종류별 필터

### 트윗 선택하기

마음에 드는 트윗을 클릭하면 오른쪽 선택 패널에 쌓입니다.  
선택한 트윗은 페이지를 새로고침해도 유지됩니다.


## 내보내기

원하는 트윗을 골랐으면 화면 아래쪽 바에서 내보내기를 선택할 수 있습니다.  
트윗이 많을 때, 여러 개의 게시글로 분할해서 내보낼 수 있습니다.


## 자주 묻는 것들

**Q. 인터넷이 없어도 됩니까?**  
→ 네, 폰트 파일 외에는 인터넷이 없어도 정상 동작합니다.

**Q. 내 트윗이 외부로 전송됩니까?**  
→ 아니요. 모든 데이터는 내 컴퓨터 안에서만 처리됩니다.

**Q. `start.bat`을 눌렀는데 창이 바로 닫힙니다.**  
→ Node.js가 설치되어 있지 않은 경우입니다. [nodejs.org](https://nodejs.org)에서 설치 후 다시 시도해 주세요.

**Q. 트윗 데이터를 못 찾겠다고 나옵니다.**  
→ `data/` 폴더가 YAYO tweet manager 폴더와 나란히 있는지 확인해 주세요.
